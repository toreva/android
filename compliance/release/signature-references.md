# Release Signature References

This file defines where signature evidence is expected for delayed public mirror
releases of `android_shadow`.

## Required references
- Artifact subject: `app/build/outputs/apk/release/app-release-unsigned.apk`
- SBOM artifact: `sbom.spdx.json` (workflow artifact `sbom-spdx`)
- Provenance: GitHub build attestation generated by `actions/attest-build-provenance@v2`
- Signature system: Sigstore/Cosign reference (external signing authority)

## Verification examples
```sh
# Verify cosign signature (replace values for your release)
cosign verify-blob \
  --signature app-release-unsigned.apk.sig \
  --certificate app-release-unsigned.apk.pem \
  app-release-unsigned.apk

# Verify GitHub provenance attestation
gh attestation verify app/build/outputs/apk/release/app-release-unsigned.apk \
  --repo "${GITHUB_REPOSITORY}"
```
